# Авторег

**Авторегистрация в СС14, использующая сервис rapidapi.**

# Установка

```shell script
git clone https://github.com/NeverMind92/Autoreg
cd Autoreg
pip install -r requirements.txt
```

# Использование

```shell script
py .\autoreg.py #Or python .\autoreg.py depending on your version of python
```
